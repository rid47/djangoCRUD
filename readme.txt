To run the project: 
(1) Create a DB : name it as "contacts"
(2) Activate venv
(3) Run migrations: python manage.py makemigrations
                    python manage.py migrate
(4) Run project: python mange.py runserver


IF you use pycharm IDE:
(1) Go to VCS ->  Checkout from Version Control -> Git
(2) Provide url ; test the connection & pull
(3) Create a DB : name it as "contacts"
(4) Run migrations: python manage.py makemigrations
                    python manage.py migrate
(5) Run project: python mange.py runserver



In case of any issue Delete The existing venv &

(1) Create a new virtual environment
(2) Install Django: pip install django
(3) Install Simplejson: pip install simplejson
(4) Install Mysql client: pip install mysqlclient-1.4.2-cp37-cp37m-win32.whl
(5) Create a DB : name it as "contacts"
(6) Run migrations: python manage.py makemigrations
                    python manage.py migrate
(7) Run project: python mange.py runserver
