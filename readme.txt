To run the project:
(1) Create a new virtual environment
(2) Install Django: pip install django
(3) Install Simplejson: pip install simplejson
(4) Install Mysql client: pip install mysqlclient-1.4.2-cp37-cp37m-win32.whl
(5) Create a DB : name is contacts
(6) Run migrations: python manage.py migrate
(7) Run project: python mange.py runserver
